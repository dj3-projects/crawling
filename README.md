# crawling
네이버 쇼핑에서 상품명, 가격, 사이트 주소를 크롤링 하여 엑셀에 저장


## 특징
* 검색어 입력시 해당 검색어로 상품 정보(상품명, 가격, 사이트 주소) 크롤링
* 상품 정보(상품명, 가격, 사이트 주소)를 엑셀에 저장
* 파일명을 crawling_(검색어)로 저장
